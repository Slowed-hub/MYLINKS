⛏ GUIDE — ONGLET MINING
▶ À QUOI ÇA SERT ?
Cet onglet permet à toute l'organisation de suivre le stock de minerais en temps réel. Tout le monde voit le même stock, et n'importe qui peut le mettre à jour.
▶ SE CONNECTER
Clique sur le bouton Connexion Discord en haut à droite. C'est obligatoire pour modifier les stocks.
▶ LIRE LE TABLEAU
MINERAI — nom du minerai
TIER — catégorie (TIER 1 = rare, TIER 3 = commun, GEMME)
STOCK TOTAL — quantité totale disponible
800+ — quantité de qualité supérieure à 800
900+ — quantité de qualité supérieure à 900
TYPE D'UNITÉ — SCU pour les minerais, unités pour les gemmes
▶ METTRE À JOUR UN STOCK
1. Trouve le minerai dans le tableau
2. Clique sur + (vide) ou ✎ (déjà renseigné) à droite
3. Saisis les nouvelles quantités en 800+ et 900+
4. Clique ✓ VALIDER
⚠ Maximum 1000 SCU (ou unités) par champ. Si tu remets un minerai à 0, une confirmation te sera demandée pour éviter les erreurs.
▶ LES FILTRES
TOUS / TIER 1 / TIER 2 / TIER 3 / GEMMES — filtrer par catégorie
800+ / 900+ — voir les minerais avec du stock de cette qualité
STOCK — voir uniquement les minerais renseignés
Les filtres se combinent — ex : TIER 1 + 800+ montre les T1 qui ont du stock 800+
La barre Rechercher fonctionne en même temps que les filtres.
Reclique sur un filtre actif pour le désactiver.
▶ TRIER LES COLONNES
Clique sur l'en-tête d'une colonne pour trier le tableau :
MINERAI — tri alphabétique A→Z / Z→A
TIER — tri par catégorie (T1, T2, T3, GEMME)
STOCK TOTAL — tri par quantité totale
800+ — tri par quantité de qualité 800+
900+ — tri par quantité de qualité 900+

La flèche ↕ indique qu'une colonne est triable.
La flèche ▲ ou ▼ indique le tri actif et son sens.
Reclique sur la même colonne pour inverser l'ordre.
▶ LES LOGS
Clique sur LOGS pour voir l'historique de toutes les modifications — qui a changé quoi, quand.
